# cnwebs-fifa2026-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面记录了与 2026 年 FIFA 世界杯相关的各类信息、笔记或辅助工具，以静态网页形式呈现，不依赖后端服务。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页/导航页（可选）
├── pages/             # 存放主要 HTML 页面
│   ├── example1.html
│   └── example2.html
└── assets/            # 公共资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立 HTML 页面均放置于此目录，每个页面可包含独立的样式与脚本。
- `assets/`：存放页面共享的资源文件，便于统一管理和引用。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面内容聚焦于 2026 年 FIFA 世界杯相关主题，如赛事数据、日程参考、观赛笔记等。
- 页面之间无强依赖关系，可单独查看或分享。

## 维护说明

- 欢迎通过 Issues 提出建议或报告问题。
- 如需添加新页面，请遵循现有目录结构，并在 `pages/` 下创建新文件。
- 修改或新增页面后，请更新 `index.html`（如有）以保持导航链接有效。
- 本仓库仅用于静态内容归档，不涉及动态交互或用户数据处理。
